Safety and privacy rules

Personal beauty advice with clear boundaries

PaletteMe is designed to help with color decisions, not to judge appearance, diagnose skin, or pressure users to buy.

Consent first Future photo analysis should only run after explicit user permission.
Local-first memory The prototype keeps shelf and profile data in local browser storage.
No medical claims Skin irritation, allergy, and dermatology questions should be redirected to professionals.
Shopping guardrails The agent flags duplicates and risky shades instead of pushing every new product.

  2. Memory: PaletteMe records the user's existing eye shadow, blush, lip, and base products.
  3. Agent reasoning: it connects undertone, style, occasion, and owned products into one look.
  4. Guardrail: before buying, it checks whether a shade is suitable or already duplicated.
